VISION Parser
VISION Parser 是一款极简主义的网页视频解析工具。它采用了沉浸式的暗黑风格设计（VISION UI），旨在为用户提供纯净、无干扰的观影辅助体验。本项目完全开源，基于纯前端技术栈构建。
✨ 特性 (Features)
极致设计：采用类似 Netflix 的 VISION 暗色主题，视觉舒适。
响应式布局：完美适配桌面端、平板和移动端设备。
极简操作：一键解析，支持回车键快速播放，Esc 键一键清空。
纯净体验：移除冗余线路，内置稳定解析接口。
零依赖：纯 HTML/JS 实现，无需后端服务器，通过 CDN 加载 Tailwind CSS。
🚀 快速开始 (Quick Start)
本地运行
克隆或下载本仓库。
直接双击打开 index.html 文件即可在浏览器中运行。
在线部署 (推荐)
本项目非常适合部署在 GitHub Pages：
Fork 本仓库。
进入仓库的 Settings -> Pages。
在 Source 下选择 main 分支并保存。
几分钟后，你将获得一个免费的在线解析网站。
🛠️ 配置说明 (Configuration)
本项目核心逻辑位于 index.html底部的 <script> 标签内。
如果你想更换解析接口，请修改 API_URL 变量：
// 修改此处的 URL 即可更换解析源
const API_URL = "[https://im1907.top/?jx=](https://im1907.top/?jx=)";
